# Koa-Swagger-Restful-Api-Mysql-Demo

this is Koa Swagger Restful Api Mysql template demo

git clone the code  

chang the db config and url config in config/config.js


### Begin to Run 
```bash
npm install
npm run dev

open  http://localhost:3008/swagger-html
```
